# زمان‌بندی انتشار وبلاگ

دستور:

```bash
python manage.py stagger_blog_publish
```

## منطق

- **۲ پست در روز** از امروز به عقب
- **محبوب‌تر = جدیدتر** (لیست در `core/seed_data/blog_publish_order.py`)
- ساعت‌ها: حدود ۱۰:۳۵ و ۱۶:۲۷ (با کمی پراکندگی)
- پست‌های آینده از `plan_200.json` بعد از ۳۳ پست فعلی اولویت می‌گیرند

## تصاویر

- ۳۳ کاور در `static/img/blog/covers/{slug}.jpg`
- تم بر اساس کشور + موضوع (ویزا، بورسیه، غذا، …)
- `seed_blog_posts --force` اول کاور اختصاصی را استفاده می‌کند

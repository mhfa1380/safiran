# فهرست تصاویر مورد نیاز

این فایل تصاویری را که باید در پروژه قرار گیرند فهرست می‌کند.

## تصاویر اصلی (`static/img/`)

| فایل | استفاده |
|------|---------|
| `favicon.png` | آیکون تب مرورگر |
| `logo.png` | لوگوی موسسه در هدر و فوتر |
| `learning_img.png` | تصویر بخش «درباره ما» |
| `single_cource.webp` | صفحه جزئیات دوره |
| `banner_bg.png` | پس‌زمینه بنر (style.css) |
| `banner_img.png` | تصویر بنر (style.css) |
| `about_overlay.png` | overlay بخش درباره ما |
| `learning_img_bg.png` | پس‌زمینه بخش یادگیری |
| `advance_feature_bg.png` | پس‌زمینه بخش خدمات |
| `quote.png` | آیکون نقل‌قول |
| `breadcrumb.png` | پس‌زمینه breadcrumb |

## تصاویر وبلاگ (`static/img/blog/`)

| فایل | استفاده |
|------|---------|
| `blog_1.png`, `blog_2.png`, `blog_3.png` | کارت‌های اخبار در صفحه اصلی |
| `single_blog_1.png`, `single_blog_2.png`, `single_blog_4.png` | صفحات وبلاگ و دانشگاه‌ها |
| `author.png` | تصویر نویسنده در single_blog |

## المان‌ها (`static/img/elements/`)

| فایل | استفاده |
|------|---------|
| `primary-check.png`, `success-check.png`, `disabled-check.png` | چک‌باکس‌ها |
| `primary-radio.png`, `success-radio.png`, `disabled-radio.png` | رادیو باتن‌ها |
